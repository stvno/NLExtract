**********************
Downloads - Instructie
**********************

Om gebruik te maken van NLExtract data, gedownload via
de `Geotoko.nl Website <https://geotoko.nl>`_, is
ondersteunende software nodig. Dit zijn de tools om met de data te kunnen werken.
Hierbij is het uiteraard afhankelijk wat je wilt bereiken. In het algemeen kun je stellen dat de meeste gebruikers
gewoon de data willen gebruiken. Dan volstaat een database dump en uiteraard de database software. Met behulp van
afbeeldingen wordt dit proces stap voor stap uitgelegd voor Windows-gebruikers.
